# Introduction to Programming

Coursework and practical assignments for the *Introduction to Programming* course at UBA (Universidad de Buenos Aires).

## Overview

This repository contains exercises, algorithms, and group projects developed during the course. The main focus of the subject is problem solving through algorithmic thinking, formal reasoning, and introductory programming concepts.

## Topics Covered

- Problem specification and analysis
- Algorithm design
- Imperative programming
- Functional programming concepts
- Debugging and testing
- Basic data structures
- Program correctness and reasoning
- Development tools and workflows

## Purpose

The goal of this repository is to document my learning process while building a strong foundation in computer science and programming fundamentals.

## Technologies

- Python / Haskell 
- Git
- Linux development environment

## Notes

This repository includes academic work, experiments, exercises, and collaborative assignments completed during the course.
